- CE-1.0.3之后的版本都将支持在线配置数据库连接信息，同时自动安装数据库。
- 注意在线配置仅适用于第一次安装，重复安装会导致数据库被覆盖。

- 默认密码：`love2025`或`LikeGirlCE`
- 请认真填写安全码 尽量设置的`复杂难以猜测` 修改密码等敏感信息需输入安全码
